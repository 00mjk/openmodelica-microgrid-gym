class openmodelica_microgrid_gym.agents.Agent(history=<openmodelica_microgrid_gym.env.recorder.EmptyHistory object>, env=None)[source]¶

Bases: object

Abstract base class for all Agents. The agent can act on the environment and observe its result. This class is aims to wrap the whole learning process into a class to simplify the implementation.

This class is strongly inspired by the tensorforce library.

Parameters

  • history (EmptyHistory) – used to store agent internal data for monitoring

  • env (Optional[ModelicaEnv]) – reference to the environment (only needed when used in internal act function)

class openmodelica_microgrid_gym.agents.SafeOptAgent(mutable_params, abort_reward, kernel, gp_params, ctrls, observation_action_mapping, history=<openmodelica_microgrid_gym.env.recorder.EmptyHistory object>, env=None)[source]¶

Bases: openmodelica_microgrid_gym.agents.staticctrl.StaticControlAgent

Agent to execute safeopt algorithm (https://arxiv.org/abs/1509.01066) to control the environment by using auxiliary controllers and Gaussian process to adopt the controller parameters (mutable_params) to safely increase the performance.

Parameters

  • mutable_params (Union[dict, list]) – safe inital controller parameters to adopt

  • abort_reward (int) – factor to multiply with the initial reward to give back an abort_reward-times higher negative reward in case of limit exceeded

  • kernel (Kern) – kernel for the Gaussian process unsing GPy

  • gp_params (Dict[str, Any]) – kernel parameters like bounds and lengthscale

  • ctrls (Dict[str, Controller]) – Controllers that are feed with the observations and exert actions on the environment

  • observation_action_mapping (dict) – form controller keys to observation keys, whose observation values will be passed to the controller

:param history:Storage of internal data :type env: Optional[ModelicaEnv] :param env: reference to the environment (only needed when used in internal act function)
