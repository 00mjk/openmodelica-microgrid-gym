omg.aux_ctl.base¶

class openmodelica_microgrid_gym.aux_ctl.base.DDS(ts, dds_max=1, theta_0=0)[source]¶

Bases: object

Implements a basic Direct Digital Synthesizer (DDS) controller. Basically is a resetting integrator to provide a theta reference at a given frequency.

Parameters

  • ts (float) – Sample time

  • dds_max (float) – The value at which the DDS resets the integrator

  • theta_0 (float) – The initial value of the DDS upon initialisation (not reset)

reset()[source]¶

Resets the DDS integrator to 0

step(freq)[source]¶

Advances the Oscilator

Parameters

freq (float) – Absolute frequency to oscillate at for the next time step

Return theta

The angle in RADIANS [0:2pi]
