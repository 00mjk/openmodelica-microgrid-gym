omg.auxiliaries.pi_controllers¶

class openmodelica_microgrid_gym.auxiliaries.pi_controllers.MultiPhasePIController(PI_param, ts)[source]¶

Bases: object

Implements a number of PI controllers for use in multiphase systems Number of phases is set to N_phase = 3

Parameters

  • PI_param (PI_params) – The PI_Parameters object with the PI controller parameters (kP, kI, kB for the gains of the proportional, integral and anti-windup part and the limits of the output)

  • ts (float) – Sample time

reset()[source]¶

Resets all controllers

step(SP, CV)[source]¶

Performs a controller step calculating the error itself using the array of Setpoints (SP) and Controlled Variables (CV, feedback)

Parameters

  • SP (ndarray) – Floats of setpoints

  • CV (ndarray) – Floats of system state to be controlled (feedback)

Return output

An array of the controller outputs.

ndarray

  • ts (float) – Sample time
